# Machine Monitoring System (Streamlit)
This project is a web-based machine monitoring system that uses machine learning to predict machine failure risk based on operational data.

## Dataset
AI4I 2020 Predictive Maintenance Dataset
Link : https://www.kaggle.com/datasets/stephanmatzka/predictive-maintenance-dataset-ai4i-2020

## 🎯 Goal
Build an intelligent system that:
- Predicts machine failure
- Assigns risk scores
- Supports monitoring decisions

## 🛠️ Tech Stack
- Python
- Pandas
- Streamlit
- Scikit-learn (coming next)

## 🚀 Features (Current)
- Streamlit web app
- Dataset overview page
- Target distribution visualization
- Data understanding and preprocessing setup

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
